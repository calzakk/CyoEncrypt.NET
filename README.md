# CyoEncrypt.NET

CyoEncrypt.NET is an easy-to-use command-line file encryptor.

## Features

- Written in .NET 5.0
- CyoEncrypt can encrypt (and decrypt) a single file
- Files are encrypted using AES
- Encrypted files are identified with an extension of ".encrypted"
